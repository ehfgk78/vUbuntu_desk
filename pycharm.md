#pycharm
---

## Install
```
 $ sudo add-apt-repository ppa:mystic-mirage/pycharm

 $ sudo apt-get update

 $ sudo apt-get install pycharm
```

## How to run 
```
 $ pycharm-community
```

## remove
```
 $ sudo apt-get remove pycharm-community
 
 $ sudo add-apt-repository-remove ppa:mystic-mirage/pycharm
